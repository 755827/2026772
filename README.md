# ARtoy — AR Demo 小程序

简要说明

- 这是一个微信小程序 AR 示例项目，包含多个 AR 场景（如 `washer-ar`）。
- 在微信开发者工具中打开项目根目录进行预览和调试。

运行步骤

1. 安装依赖（如果有本地构建脚本）：

```powershell
# 可选，项目如有构建或依赖安装
npm install
```

2. 打开微信开发者工具，导入本项目根目录 `d:/ARtoy/artest_v01首页完成`，选择小程序类型（测试号/体验版需填写相应 appid）。

注意

- 已在 `.gitignore` 中排除了 `project.private.config.json` 等敏感/本地配置文件。
- 若需设置仓库访问权限或添加协作者，请在 GitHub 仓库页面（https://github.com/755827/2026772）中操作。
